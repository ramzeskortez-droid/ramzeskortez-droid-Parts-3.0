
# История версий проекта

## 1.6.9
- **Seller UI:** Переписана функция `parseRuDate`. Теперь она корректно обрабатывает строки с переносами, лишними пробелами и специфическими разделителями из Google Sheets.
- **Logic:** В расчет `marketStats` добавлена проверка на верхнюю границу времени (`endOfToday`). Это исключает попадание будущих или ошибочных дат в статистику текущего дня.
- **Dashboard:** Статистика «Сегодня/Неделя/Месяц» теперь строго ориентирована на системные сутки.

## 1.6.8
- **Client UI:** Исправлена проблема с узкой колонкой статуса. Сетка расширена для корректного отображения бейджей «В ОБРАБОТКЕ».
- **Seller UI:** Восстановлены дашборды «НЕДЕЛЯ» и «МЕСЯЦ». Карточка «Лидер спроса» перенесена на отдельную строку для лучшей читаемости.
