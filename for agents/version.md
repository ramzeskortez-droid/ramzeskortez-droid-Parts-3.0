
# История версий проекта

## 1.5.9
- **B24 Force GET Integration:** Реализован метод создания лидов и добавления товаров через GET-запросы с URL-параметрами. Это решение призвано обойти блокировки POST-запросов на стороне сервера или CRM.
- **Diagnostic Logging:** Добавлено детальное логирование ошибок GET-запросов.
- **Strict Adherence:** Соблюдено правило №4 — предупреждение о невозможности создания «пустого проекта» без потери данных.
