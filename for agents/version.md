
# История версий проекта

## 1.7.0
- **Client UI:** Добавлена обязательная авторизация для клиентов. Реализовано модальное окно с полями Имя/Телефон и кнопками быстрого входа "Демо Клиент 1" и "Демо Клиент 2".
- **Auth:** Сессия клиента (`client_auth`) хранится независимо от сессии поставщика (`seller_token`). Добавлена кнопка выхода (Logout) в шапку клиента.
- **UX:** При создании заказа имя клиента автоматически подставляется из данных профиля.

## 1.6.9
- **Seller UI:** Переписана функция `parseRuDate`. Теперь она корректно обрабатывает строки с переносами, лишними пробелами и специфическими разделителями из Google Sheets.
