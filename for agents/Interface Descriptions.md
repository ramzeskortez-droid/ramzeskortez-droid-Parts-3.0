# Описание интерфейсов и элементов управления (UI Dictionary)

## 1. Интерфейс Клиента (Client Interface)

### 1.1 Модальное окно авторизации
*   **Поле "Имя"**: Текстовое поле. Автоматически приводится к верхнему регистру (`uppercase`). Обязательное.
*   **Поле "Номер телефона"**:
    *   *Поведение:* Ввод цифр форматируется в `+7 (XXX) XXX-XX-XX`.
    *   *Валидация:* Если длина < 18 символов, поле подсвечивается красным, кнопка "Создать аккаунт" неактивна. При вводе лишних цифр они просто игнорируются (не сбрасывая поле).
*   **Кнопка "Создать аккаунт"**: Активна только при заполненном имени и корректном телефоне.

### 1.2 Хедер и Профиль
*   **Блок профиля**: Имя + Телефон.
*   **Кнопка "Выход"**: Сброс сессии.

### 1.3 Форма создания заказа (New Order Form)
*   **Поле "VIN / Шасси"**: Текстовое, `uppercase`.
*   **Секция "Автомобиль"**:
    *   **Combobox "Марка (Бренд)"**:
        *   Текстовое поле с выпадающим списком.
        *   При вводе текста список фильтруется (например, "Au" -> "Audi", "Aurus"...).
        *   Выбор кликом подставляет значение.
        *   *Валидация:* Если введен текст, которого нет в справочнике, поле подсвечивается красным ("Выберите марку из списка").
    *   **Поле "Модель"**: Текстовое поле. Ввод любой модели вручную (например, "X5 G05").
    *   **Поле "Год"**: Текстовое.
*   **Блок "Позиции заказа"**:
    *   **Название**: макс 90 символов.
    *   **Количество**: число, макс 1000.
*   **Кнопка "Отправить запрос"**: Создает заказ.
